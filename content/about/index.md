Mình là **Phạm Đình Hiệp**, một lập trình viên thích cách làm việc rõ ràng: hiểu bài toán trước, chọn giải pháp vừa đủ, và tối ưu dần dựa trên dữ liệu.

## Mình tập trung vào

- **Backend**: Java (Core Java, nền tảng cho Spring Boot)
- **Frontend**: JavaScript (ES6+), tư duy async và tương tác DOM
- **Network programming**: TCP/IP, Socket, HTTP (để hiểu hệ thống “chạy thật”)
